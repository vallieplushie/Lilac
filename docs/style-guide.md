# Lilac Style-guide

## Indentation

Soft, 4 space indents. Banner indentation is best:

```
I number: 12
B fizzbuzz: (I num) => (
    B isBig: false
    num > 10 ? (
        print "fizz"
        isBig: true
        )
    |
    num < 10 ? (
        print "buzz"
        )
    isBig
    )
fizzbuzz number
```
